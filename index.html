Libraries

  • three.js - 3D library
  • minivents.js - event system
  • jQuery - JavaScript toolkit
  • tween.js - tweening engine
  • dancer.js - audio API
  • Layers CSS - CSS framework

3D models

  • Angel statue - Lucy A Christian Angel Statue by anavriN
